# 란도랜드2

참가자가 직접 만든 이야기가 뉴스와 주가에 반영되는 모의 주식시장 리그입니다.

현재는 주요 기능을 체험할 수 있는 데모 리그가 운영됩니다.

## 참가하기

1. 카카오 계정으로 로그인합니다.
2. 리그에 참가하면 중복 없는 익명 닉네임이 자동으로 정해집니다.
3. 초기 자산 1,000,000 RP를 받고 거래를 시작합니다.

카카오 계정 하나로 한 번만 참가할 수 있으며, 이메일 제공 동의는 요청하지 않습니다.

## 거래 방식

- 하루가 한 라운드이며 모든 일정은 한국 시간 기준입니다.
- 주문은 언제든 접수할 수 있고, 정산 전까지 자유롭게 취소할 수 있습니다.
- 매일 오전 9시에 대기 주문이 **변동 전 가격**으로 일괄 체결됩니다.
- 주문 체결이 끝나면 그날의 뉴스와 새로운 주가가 공개됩니다.
- 한 라운드에 매수와 매도 주문을 각각 최대 5건까지 접수할 수 있습니다.
- 본인이 상장한 종목은 직접 거래할 수 없습니다.

## 종목 상장

참가자는 리그마다 종목을 하나까지 상장할 수 있습니다. 상장 없이 투자자로만 참여해도 됩니다.

상장할 때는 종목명, 최초 주가, 종목 설명, 콘셉트와 테마, 주차별 이야기를 제출합니다. 제출한 이야기는 매일 공개되는 뉴스와 시세 흐름에 활용됩니다.

## 주요 기능

- 매일 새로운 종목 뉴스와 캔들차트 갱신
- 하루 최대 ±30% 범위의 주가 변동
- 순자산의 최대 50%까지 레버리지 사용
- 매일 출석토큰 지급 및 홀짝 사다리 참여
- 매주 일요일 오전 9시 참가자 순위 공개
- 완료한 매매 횟수와 보유 라운드 자동 집계

## 순위와 칭호

순위는 현금, 보유 종목 평가액, 레버리지 비용을 반영한 순자산을 기준으로 계산합니다. 리그 종료 후에는 수익 순위와 함께 단타왕, 인간지표, 존버왕 등의 칭호가 결정됩니다.

란도랜드2의 RP와 모든 거래는 실제 금전 가치가 없는 모의 거래입니다.
